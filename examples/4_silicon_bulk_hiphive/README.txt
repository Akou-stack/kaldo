2. If you have not yet complied lammps as python library (please see the documentation for this set up), please run the following

   commands in orders to run the example from the scratch:
   
   python 1_prepare_reference_structures.py
   python 2_build_structure_container.py
   python 3_compute_fcs.py
   python 4_calculate_phonons.py

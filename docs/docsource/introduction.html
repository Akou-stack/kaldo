    The kALDo package¶

    kALDo. Thermal transport in crystalline and non-crystalline solids.

    kALDo features real space QHGK calculations and three different solvers of the linearized BTE: direct inversion, self-consistent cycle and RTA. Below we illustrate the main features of the code
